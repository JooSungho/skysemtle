# skysemtle

This repository stores a recovery-friendly three-day routine scheduled in Korea Standard Time (Asia/Seoul). The plan highlights hydration, nutrient-dense meals, gentle movement, and consistent rest to support a gradual return to daily activities.

## Files

- `schedule.json`: Structured timeline containing each activity with timestamps, reminders, and notes for the three-day recovery plan.

## Usage

Import `schedule.json` into your preferred calendar or wellness tracking tool to keep these reminders handy. Each entry includes:

- `id`: Unique identifier for the reminder.
- `datetime`: ISO 8601 timestamp with timezone offset.
- `title`: Short description of the activity.
- `notes`: Additional instructions or context.
- `repeat`: Recurrence rule, set to `none` for these one-off reminders.

Feel free to adapt the schedule to match your personal recovery pace and consult with a healthcare professional for tailored advice.
